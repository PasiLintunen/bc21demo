# bc21demo
Code Bootcamp 2021 Demo

This web singlepager application was created to maintain multiple moviecollections. 

It is powered by
----------------

Cloudynary (CDN),
ElephantSQL (Postgres),
Netlifly,
Heroku,
The movie database

Technologies used:
-----------------

Backend
-------

Python 3,
Flask,
SQLAlchemy

Frontend
--------

HTML5,
CSS3,
Javascript,
Jquery 3,
Bootstrap 4
